# 📊 Demographic Data Analyzer

A Python project built using **Pandas** to analyze demographic data from the **1994 U.S. Census Database**. This project is part of the **freeCodeCamp Data Analysis with Python Certification**.

## 🚀 Features

The program analyzes the dataset and answers the following questions:

- 👥 Count of people from each race.
- 👨 Average age of men.
- 🎓 Percentage of people with a Bachelor's degree.
- 💰 Percentage of people with higher education earning more than \$50K.
- 📉 Percentage of people without higher education earning more than \$50K.
- ⏰ Minimum number of working hours per week.
- 💵 Percentage of people earning more than \$50K among those working the minimum hours.
- 🌍 Country with the highest percentage of people earning more than \$50K.
- 💼 Most common occupation in India among people earning more than \$50K.
